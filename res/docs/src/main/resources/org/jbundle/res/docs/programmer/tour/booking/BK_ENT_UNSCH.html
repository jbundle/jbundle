/PROG_HDR/

Program Description:

Enter a unscheduled tour.

/SCREEN1/

Program Operation:

This program is called from some other module and may be provided the Tour header, the Departure date, or both.

If the Tour header has not been entered, then you can enter the tour code and the description will display.

If the departure date hasn't been entered, then this can be entered here also.

The program will display the tour information and the current inventory for this departure date.

If questions exist for this tour, then the screen displays.
