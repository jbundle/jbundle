/PROG_HDR/

Program Description:

This program is used to display tours by the tour type.

/SCREEN1/

Program Operation:

Enter the tour type to display and the starting and ending departure dates.

All tours with this tour type will display.

The valid tour types are:

G - Group Series Tour

U - Unscheduled Tour

M - Modular Tour

F - Fit Tour

M - Modular Component Header

C - FIT Component Header
