/PROG_HDR/

Program Description:

This program is used to review and allow changes to the questions and answers selected for this tour.

/SCREEN1/

Program Operation:

Each answer selected for this tour will display on the screen. If the answer has a "Y" in the entered column, then it means you answered this question. A "N" means the question was selected automatically.

You will be asked if you want to clear the current answers and re-enter new answers. If you answer yes and you have set up any of the detail information, the system will ask you if you want to clear the current detail. All the pricing, hotels, land, air, remarks, and comments must be blank before selecting new answers.

After clearing the current detail, you can select a new set of answers for this booking.
