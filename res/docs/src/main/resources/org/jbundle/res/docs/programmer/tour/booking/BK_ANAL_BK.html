/PROG_HDR/

Program Description:

This program organizes and prints booking data by reservation agent.

/SCREEN1/

/SCREEN2/

Program Operation:

Decide whether to print in report or spreadsheet format. Spreadsheet format will not print, but will output the file into the :PRINT directory under the name SSHEET_XXXX_USERNAME. This file may be retrieved in standard spreadsheet format.

Enter a starting and ending booking date. Selecting a reasonable range will speed up the analysis considerabilly.

Choose the date to analyze by. The report will be ordered by reservation agent, then this date. The date descriptions are:

B - Booking date

A - Acceptance date (Status -> OK or money received)

D - Deposit received date

F - Final payment received date

D - Departure date

M - Last modification date (includes cancelled bookings)

Enter a date range for the date type selected.

If you want to include all modification, rather than just the initial modification, answer Yes to the last question. This may be useful in analyzing data on who, when and how many times bookings are modified from beginning to end.

The report or spreadsheet will print automatically.
