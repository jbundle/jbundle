/PROG_HDR/

Program Description:

This program is used to display tour detail for any tour without having to book the tour.

/SCREEN1/

Program Operation:

Enter a tour code and target departure date (They will default to the current tour and departure). All the possible questions for this tour will display. The main tour information is usually under the first question.

/SCREEN2/

Select a question to display all the possible answers. All the possible answers to the question will display.

/SCREEN3/

Next, the answer information menu will display. Select an item to display the pricing, hotel, land, air, or remark information for.

/USE: BK_ALT_PRICE/

/USE: BK_ALT_HOTEL/

/USE: BK_ALT_LAND/

/USE: BK_ALT_AIR/

/USE: BK_ALT_REMARK/
