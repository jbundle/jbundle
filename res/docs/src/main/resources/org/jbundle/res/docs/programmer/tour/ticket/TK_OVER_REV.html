Program Description:

This program is used to review the override received from a carrier for each ticket.

/SCREEN1/

/SCREEN2/

Program Operation:

Enter the carrier code and the starting and ending TICKET departure date to process overrides receipts for. If you want to review the override received on a particular date, enter the date, otherwise type hit return to print all overrides received in the date range.

All the submitted tickets with override amounts entered in the range for the carrier selected will print in ticket departure date order.
