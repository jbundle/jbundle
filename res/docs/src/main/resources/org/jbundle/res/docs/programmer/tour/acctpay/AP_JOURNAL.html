/PROG_HDR/

Program Description:

This program prints the current account status for a range of vendors. Optionally, estimates of future costs can also be included in the print-out. You can also print ALL open items if you want (by default only the items with a non-zero balance will print).

/SCREEN1/

/SCREEN2/

Program Operation:

Enter the starting and ending vendor to print-out.

If you select current estimates to print-out, all the departure estimates and finalization estimates will be included on the report. The invoice balance will be non-zero for a finalization estimate which is confirmed. The USD balance for finalization estimates will always be zero.

To have the system print all the line items, including ones with zero balances, answer Yes to the last question.
