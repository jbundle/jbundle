/PROG_HDR/

Program Description:

This program is used to print vouchers pending.

/SCREEN1/

/SCREEN2/

Program Operation:

The answer to the first question will decide what types of vouchers to process; Vouchers requested, Vouchers Re-requested, or Cancellation vouchers.

Next, enter the cutoff date. All vouchers request to this date will print.

Decide whether to print hotel or tour vouchers, then enter the document type and the form to use (blank for the default form).
