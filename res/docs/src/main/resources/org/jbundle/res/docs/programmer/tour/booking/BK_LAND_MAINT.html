Program Description:

This program maintains the land file.

/SCREEN1/

/SDESC1/

Program Operation:

The land file maintenance program is a simple file maintenance program.

First, type in the land code. It is usually coded like this:

First 3 letters: City code

Next letter : Type of land entry (ie., S-Sightseeing)

Next 2 letters : Sequence to keep codes unique

Next letter : Vendor initial if applicable

For example, the code: HKGS1P would be Hong Kong Sightseeing, provided by the "P" operator.

The land description can take up to two full lines. The country and city codes should be the ARC airport codes, or three letter abbreviation of the city if there is no airport.
