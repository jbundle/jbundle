/PROG_HDR/

Program Description:

This program prints the General Ledger Worksheet.

/SCREEN1/

/SCREEN2/

Program Operation:

This program prints out a work sheet with ample space for entering adjusting (accruing) entries.

The operation of the Worksheet Print-out is exactly the same as the Trial Balance except that on the print-out there is room to make adjustments and to transfer the adjustments to a column which will be used to set-up the Financial Statements.
