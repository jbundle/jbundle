/PROG_HDR/

Program Description:

This program is used to display tickets by their status.

/SCREEN1/

Program Operation:

Enter the ticket status code and the starting booking departure date to display.

All the tickets with this status will display in booking number order. Header tickets with a matching status will display the number of passengers in the booking.

For example, the ticketing department could use this program to see which tickets are ready for ticketing (update in booking entry or at the ticketing finalization date).
