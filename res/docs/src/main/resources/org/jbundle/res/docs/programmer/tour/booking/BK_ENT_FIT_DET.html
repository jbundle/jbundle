PROGRAM DESCRIPTION:

This program is used to quickly build an FIT from scratch.

PROGRAM OPERATION:

After entering the FIT header, just go to this screen. Enter free-format text in the first line. The text can be air, land, transportation, a hotel, or a FIT package header. The format of the text dictates the usage of the field. For example entering:

AA427Y - will prompt you for the rest of the air info HKGPR - will prompt you for hotel info

HKGS1VP - will prompt you for land info

HKGMPA5 - will display the hotel, land, and air for this package (This is an FIT package)

The hierarchy goes as follows:

(First) 1. FIT packages

2. Hotels

3. Land

4. Air

5. WP paragraphs
