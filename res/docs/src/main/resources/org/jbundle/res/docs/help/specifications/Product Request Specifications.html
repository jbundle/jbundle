        • Product

        Product is any item for sale to the customer. This could include Packaged Tours, Air Seats, Hotel Rooms, Cars, Cruise cabins, or Land.

        Product Profile

        Product Profile is an object used to lookup products that meet a certain criteria.

        Interface:

        AddLocation - (Country/Region/City/Airport/etc..)

        addVendor

        addProductType - (Tour, Air, Hotels, Cars, Cruises, or Land)

        setClass - Luxury, Superior, Standard, etc

        setRateType - AdHoc, Group, Rack, etc...

        getNextProduct - Used to read through the products that match this profile.
