/PROG_HDR/

Program Description:

This sub-menu is used for setting up the payments and the proper distribution of the payments.

/SCREEN1/

Program Operation:

Select the program which you would like to use.

The items with the "*" next to them are the programs that you will use most. The others are there for your convenience.

NOTE: If you type All at the "CHANGE ITEM" prompt, all invoices will be selected for their current balance.
