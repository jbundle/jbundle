/PROG_HDR/

Program Description:

This program is used to edit the latest itinerary for this booking without re-building it.

/SCREEN1/

Program Operation:

When you enter this program, you are put directly into the word processor, editing the latest version of the itinerary.

If you have made changes to the itinerary since last editing it, you should rebuild it first.

This program saves time if you automatically have the tour itineraries built, and you want to print them out.
