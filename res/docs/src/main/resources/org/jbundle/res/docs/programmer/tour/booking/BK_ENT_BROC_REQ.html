/PROG_HDR/

Program Description:

This program is used to enter brochure requests.

/SCREEN1/

Program Operation:

Select an agency either by ARC number or search by name and zip (enter ARC 0 for manual entries). Then select the person to mail it to (optional), UPS or Bulk Mail, and enter the brochures to mail (defaults to All 5).

The brochure request will be recorded.

Type DEL to delete the brochure request.
