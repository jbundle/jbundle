/PROG_HDR/

Program Description:

This program allows you to display and change the tour remarks.

/SCREEN1/

Program Operation:

Remarks are the free-text information that will be printed on the invoice and other forms. Include information of use to the agent or the clients.

If the remarks already exist, then they will display.

The remarks are entered line-by-line. Enter a Description on each line.

Typing DEL at the "Change" prompt clears all the current remarks and reverts back to the original tour remarks.
