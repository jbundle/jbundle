/PROG_HDR/

Program Description:

This program is used to display the usage for a range of group series departures.

/SCREEN1/

Program Operation:

Enter a group series tour code and a starting departure date. All tours departures will display in departure date order.

Note: This program is very similar to the tour availability display (which can also be used). The difference is this display also gives you the tour status at each departure, so cancelled or blocked departures will stand out.
