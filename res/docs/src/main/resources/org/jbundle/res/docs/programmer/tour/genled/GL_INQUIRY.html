/PROG_HDR/

Program Description:

This program allows you to see the detail transactions made to a G/L account in a specified period.

/SCREEN1/

Program Operation:

First, enter the account number, and a beginning and ending date (defaults to current period). Then answer Yes if you want to see the beginning balance for the account.

Then the system will display the activity in this account during that period.

Note: If you want to see the beginning balance for an account, the system must read through all the transactions before that date to calculate the amount. This may result in a delay in the display.

If you don't display the starting balance, the ending balance is the change for the transactions displayed.
