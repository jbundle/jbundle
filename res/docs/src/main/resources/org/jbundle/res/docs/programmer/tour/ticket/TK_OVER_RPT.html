/PROG_HDR/

Program Description:

This program is used to print the report of air overrides due on departed tickets.

/SCREEN1/

Program Operation:

Enter the carrier code and the starting and ending TICKET departure date to process overrides for. If you want to include tickets which have the override paid already, answer Yes to the last question.

All the departed tickets in the range for the carrier selected will print in ticket-number order.

NOTE: The file is not updated to reflect the fact that the tickets have been submitted for payment, but when the payment is received from the carrier, the detail can be recreated by running this report again for the departure range submitted.
