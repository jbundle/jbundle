/PROG_HDR/

Program Description:

This program prints a deposit slip for all of the cash receipts in the current batch.

/SCREEN1/

/SCREEN2/

Program Operation:

The name of the current batch will display. To print the report, hit return at the CHANGE ITEM prompt.

Note: If checks are entered to be distributed to multiple bookings, they must have been entered together for this print-out to work correctly.
