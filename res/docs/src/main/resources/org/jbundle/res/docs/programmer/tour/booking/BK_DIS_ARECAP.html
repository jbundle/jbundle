/PROG_HDR/

Program Description:

This program is used to display the air recap for a range of dates.

/SCREEN1/

Program Operation:

Enter the carrier and flight number and a starting date. If you only want to view the air taken from inventory, answer yes to the inventory question.

The air recap will display in date order. Those air items which are passenger modifications are displayed with the passenger name associated with the air segment. Booking air segments display the passenger allocation (with the passenger mods deducted).

Cancelled bookings do not display.
