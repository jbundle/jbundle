/PROG_HDR/

Program Description:

This program is used to print the ticketing transaction review report.

/SCREEN1/

/SCREEN2/

Program Operation:

This program can be used as a pre-G/L posting review, or a G/L transaction source detail after posting.

First, enter the transaction type:

"I" will print issued ticket transactions

"D" will print departed ticket transactions

"O" will print ticket override transactions

Next, enter the date range to print.

To print only transaction posted to the General Ledger, answer "Y"; To print only transactions NOT posted to the G/L type "N"; otherwise press return and all transactions will print.

To print the booking description of each transaction, answer Yes to the last question.

The report prints the debit, credit and variance amounts for each transaction. If the variance is positive, it will be a debit; negatives are credits.

NOTE: Since this program is not run often, it has to do an inefficient sequential search throuh the Ticket detail. This program can take a long time to run.
