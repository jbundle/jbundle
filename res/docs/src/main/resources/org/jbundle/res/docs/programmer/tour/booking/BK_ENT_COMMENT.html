/PROG_HDR/

Program Description:

This program allows you to display and change the tour comments.

/SCREEN1/

Program Operation:

Comments are the free-text information that is used for internal purposes. Comments will never be printed on any documents to be distributed externally.

If the comments already exist, then they will display.

The comments are entered line-by-line. Enter a Description on each line.

Typing DEL at the "Change" prompt clears all the current comments. Comments can't come from the tour header.
