/PROG_HDR/

Program Description:

This program is used to re-print an invoice, change a booking's status, or print an invoice on-line.

/SCREEN1/

Program Operation:

When you enter this program, the current booking status will display.

If you change the Print Invoice flag to Yes, the booking status will change to "OK" and the invoice will automatically print on the next invoice run.

If you type a Yes into the Print invoice now field, the invoice will be printed immediately.
