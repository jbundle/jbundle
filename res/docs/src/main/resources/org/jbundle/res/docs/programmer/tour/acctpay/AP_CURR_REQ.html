/PROG_HDR/

Program Description:

This display is used to quickly estimate how much of a particular currency is required at any time. Each vendor who bills in the selected currency will display with the current amount due. The total displays at the bottom of the screen.

/SCREEN1/

Program Operation:

Enter the currency code to display.

Each vendor with the the entered currency will display with the current account balance of each.

NOTE: This display may take some time if there are vendors with many outstanding items.
