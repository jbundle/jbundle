/PROG_HDR/

Program Description:

This program is used to display land by vendor.

/SCREEN1/

/SDESC1/

Program Operation:

Enter the vendor code of the vendor to display. If you want to see land pricing, enter a passenger count and a travel date for the price.

All the land will display for the vendor. Optionally, the Seat-in-car/coach and private car prices will also display.
