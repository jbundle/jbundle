/PROG_HDR/

Program Description:

This program allows you to display and change the tour land information.

/SCREEN1/

Program Operation:

If any land detail already exists, then the current land is displayed.

The land codes are entered line-by-line. First the land code or transportation code is entered, then the tour order flag, acknowledgement days, inventory flag and land status.

Typing DEL at the "Change" prompt clears all the current land codes and reverts back to the original land for this tour.
