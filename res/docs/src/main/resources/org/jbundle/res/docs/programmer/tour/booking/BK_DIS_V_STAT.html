/PROG_HDR/

Program Description:

This program is used to display vouchers which have the same status. (For example, display vouchers waiting for acknowledgement but not yet received)

/SCREEN1/

Program Operation:

Enter a valid voucher status and a starting date to display. The vouchers of that status will display in date order.

The voucher status codes are:

0 Vouchers being held

1 No Status

2 Request Voucher be printed (Voucher printed next run)

3 Voucher Printed, Waiting for Acknowledgment (After prtd)

4 Acknowledgment Received (Voucher Confirmed)

5 Voucher Printed (No Ack Requested) (After printed)

6 Cancellation Voucher Requested (Xl Vchr prtd on next)

7 Waiting for Cancellation Acknowledgement

8 Cancellation Acknowledgement Received

9 Cancellation Voucher Printed

10 Request the voucher be re-printed
