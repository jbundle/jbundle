/PROG_HDR/

Program Description:

This is the main menu for the booking system. All the other programs are called from this menu.

/SCREEN1/

Display Programs

Program Operation:

If you haven't selected a tour yet, then this program will display the generalized menu. If a tour has been selected, the the appropriate sub-menu will automatically display.

You can select any sub-program from this menu (not just the ones displayed).

After using a program on this menu, only the select prompt appears at the bottom of the screen. If you hit return again, then the menu will redisplay. If you press return a second time, the screen will return to a blank booking screen.

Several Sub-Menus can be displayed from the booking menu.
