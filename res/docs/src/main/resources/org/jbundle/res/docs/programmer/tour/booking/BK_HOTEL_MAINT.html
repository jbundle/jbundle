Program Description:

This program maintains the hotel file.

/SCREEN1/

/SDESC1/

Program Operation:

The hotel file maintenance program is a simple file maintenance program.

First, type in the hotel code. It is usually coded like this:

First three letters: City code

Next two letters : First type letters of hotel name

Next letter : Vendor initial if applicable

The hotel name can take up to two full lines. The country and city codes should be the ARC airport codes, or three letter abbreviation of the city if there is no airport.

The last item on the screen refers to the room type that the tour escort gets when enough passengers are booked. (Twin share or Single)
