/PROG_HDR/

Program Description:

This program is used to display the modification history of a booking, and the booking detail at each stage of its history.

/SCREEN1/

Program Operation:

When you enter this program, the booking history is displayed. Each modification is displayed, along with an "X" by the detail items which were changed at the step.

The system then asks if you want to display the detail at any of the steps. If you enter a mod. number, the detail modification menu will appear.

/SCREEN2/

/SDESC2/

Select the item to display the modification's detail information.
