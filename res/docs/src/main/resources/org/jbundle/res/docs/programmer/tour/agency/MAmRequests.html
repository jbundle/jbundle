Brochure request entry

Program Description:

The Brochure request program is used to enter brochure requests.

/SCREEN1/

Program Operation:

Entering a value in the "IATA/Key" field will retrieve an Agent by their IATA number, or a Passenger by their Key depending on the value in the "Agent/Pax" popup box. To enter a manual Brochure request, tab past this field.

Pressing the lookup button will display the Agency lookup screen where you can select an agency.
Pressing the form button will display the Agency maintenance screen where you can enter, lookup, or change an Agency. Any changes to the main record is reflected in the current brochure request.

Pressing the lookup button next to the "Passenger:" label will display the Passenger lookup screen.
Pressing the form button will display the Passenger maintenance screen.

The "Send Via" popup box can be used to select the method to mail brochures, or to group brochure requests so they will not auto-print with the general brochure requests. This field defaults to the value in the Agency control file the first time this screen is entered.

The "Bundle" and "Quantity" fields can be used to select a bundle of brochures to mail. These fields also default to the Agency control file values for the first brochure request.

The Text field can be used to enter free text to appear on the brochure request label.

The Brochure Detail button brings up the Brochure Detail Display Screen where you can enter individual brochures to mail.

The Brochure History button brings up the Brochure History Display Screen which displays the brochures mailed to this agency or passenger in the past. The history screen tracks bundles and individual brochures mailed.
