/PROG_HDR/

Program Description:

This program corrects any G/L postings that are made to G/L accounts that have been deleted.

/SCREEN1/

/SCREEN2/

Program Operation:

Enter the G/L number to change any incorrect G/L accounts to. Then enter the comment to assign to any corrected entries. If you enter a blank comment, the original comments will remain unchanged.

This program scans all the current G/L detail, if any postings were done to accounts with non-existent G/L numbers, the G/L number will be changed to the G/L number entered in this screen.

Note: This program requires exclusive access to the G/L detail and Chart of accounts file while it is processing.

Note: You will usually change the account numbers to a summary account, then immediately create journal entries to post to the correct G/L accounts.
