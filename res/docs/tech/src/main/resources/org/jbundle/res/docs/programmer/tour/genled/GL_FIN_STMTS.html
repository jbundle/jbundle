Program Description:

This program prints the financial Statements.

/SCREEN1/

/SCREEN2/

Program Operation:

Use this program to print the financial statements for period-end or year-end purposes. It also allows the user to select many of several formats for printing financial statements. The program will also automatically go to the year-end procedure if the user has completed all of the required statements.

This portion of the program works as a job-processor. You enter all of the statements you would like to be printed with all of the parameters and the computer automatically processes and prints them.

The computer will ask for the following information on each line:

(1) Format number:

(2) Statement to print: (Enter statement range)

(3) Budgets or Comparatives? (or neither)

(4) Profit Center (def=consolidated)
