/PROG_HDR/

Pprogram Description:

This program is used to enter the booking tour comments. Booking tour comments may be displayed but not saved or changed in the booking entry program.

/SCREEN1/

/SDESC1/

Program Operation:

The tour header description and the current question/answer sequence will display.

Then, the current comments will display.

The sequence number is only used to replace/delete subsequent comment entries. The remaining field is transfered to the actual booking in booking entry.
