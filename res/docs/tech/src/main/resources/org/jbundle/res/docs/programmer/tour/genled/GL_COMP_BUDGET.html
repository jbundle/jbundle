/PROG_HDR/

Program description:

This module allows you to enter and modify Comparative and Budget information for use in Financial Statements.

/SCREEN1/

Program operation:

First, enter the Chart of Accounts account number, then enter a "C" or "B" to edit the comparative or budget information for this account.

Enter the period ending date and the balance for each account.

This information is used as comparasion in the financial statement printing modules.

Note: Comparative information is automatically entered into this file after the year-end financial statements have been run. (The prior-year balance for an account is part of the comparative information)
