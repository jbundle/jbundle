/PROG_HDR/

Program Description:

This program is used to display the detail for a voucher. (For example, you want to see how many nights a hotel voucher was for)

/SCREEN1/

/SCREEN2/

/SCREEN3/

Program Operation:

First, enter the tour number of the tour to display. The tour description and voucher detail will display. If the voucher is based on the current hotel or land detail, then the flag "Cur det" will be set to Yes.

Once the voucher detail displays, select a voucher to display the detail for. The hotel or land detail for that booking/modification will display.
