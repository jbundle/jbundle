/PROG_HDR/

Program Description:

This report prints each G/L account with a starting and ending balance and optionally, the account activity during the time frame.

/SCREEN1/

/SCREEN2/

Program Operation:

The General Ledger is used to get information about an accounts balance as of a certain date, or it's activity during a certain period.

When this selection is choosen, the computer will print out each General Ledger Account, along with the activity and balances for a period, and the detail of each transaction (if the option is choosen).

You will be asked for a starting and ending account, a starting and ending date, whether detail transactions should be shown, and the profit center to select (if available). The starting and ending dates default to the starting and ending dates of the current period.

Usually you will select all dates for the G/L entry date range. You should only select a range when you have to see what entries were done in a particular time-frame.

The report prints out in account order the Chart of Accounts. Along with the header information, the Beginning balance (dr/cr), net change change, and ending balance for the period choosen.
