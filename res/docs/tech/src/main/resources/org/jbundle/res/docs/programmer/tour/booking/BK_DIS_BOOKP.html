/PROG_HDR/

Program Description:

This program is used to display all reservations by passenger's last names.

/SCREEN1/

Program Operation:

This program will list all (not just the lead name) passengers by last name.

Enter a starting last name in upper case. Then enter a starting reservation date to display. The date defaults to the date as of one year ago, but any valid date may be entered.

Remember: Be aware of the booking status; cancelled bookings display also.
