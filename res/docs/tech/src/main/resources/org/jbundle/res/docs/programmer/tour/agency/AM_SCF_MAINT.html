/PROG_HDR/

Program Description:

This program keeps track of SCF codes for mailing and sales analysis.

/SCREEN1/

Program Operation:

Enter a starting SCF number; If any SCF records contain this range, they are read, otherwise you can set up a new SCF record.

Note: When entering the SCF area description, always include the state code at the end, for state sales analysis. (ie., Los Angeles, CA)
