/PROG_HDR/

Program Description:

This is the main entry screen from group-series type tours.

/SCREEN1/

Program Operation:

After entering the information on this screen, by default the system goes to the next screen.

If the passenger information has already been entered, then the "Tour Information Entry" screen will appear. All of the selected items will display along with the total. Then all of the Alternate questions will be resolved. After this, the Optional questions will be displayed and any optionals may be added.

If the passenger informations hasn't been entered, then the "Tour Pricing Display" screen will appear. Pricing for Double, Single, and Triple passengers will appear on the screen. The Alternate questions and Optional questions will also display. You also use this screen to see what the price would be for each question and answer combination.
