Program Description:

Enter a modular tour header.

/SCREEN1/

/SDESC1/

Program Operation:

If the Tour header has not been entered, then you can enter the tour code and the description will display.

If the departure date hasn't been entered, then this can also be entered here.

The program will display the tour information. Any of the values on the screen may be changed. For proposals, the passenger counts can be entered in this screen.

If questions exist for this tour, then the screen displays.
