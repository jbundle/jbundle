/PROG_HDR/

Program Description:

This program allows changes to the control file. The control file has the basic operating information for the G/L system.

/SCREEN1/

Program Operation:

The first time this program is run, you will be asked to fill in all the fields; thereafter the fields will display and you can change any fields.

The Starting and Ending dates are the range for the current period.

The accrual source code is the source code which will automatically create a reversing entry for an accrual entry.

Answer "Yes" for profit centers if you will be using profit centers in your chart of accounts. If you answer no, you may use the last three digits in the account numbers as part of the account number.
