/PROG_HDR/

Program Description:

The Agency Maintenance system is used to keep track of agencies and to manage mailing.

/SCREEN1/

Program Operation:

Agents may be retrieved either by their ARC number, or their agency name and zip code.

Changes may be made as with any other file maintenance program.

Note: This system is not accessable to the reservation agent; They can only access agencies in agency selection and brochure requests.
