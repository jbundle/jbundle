Program Description:

This program is used to print the cash receipts planning report.

/SCREEN1/

/SCREEN2/

Program Operation:

This program is used to estimate future cash receipts based on current sales.

Enter the starting date, period length (days) and number of periods.

Remember, a tour is only considered if the status is "OK" or some money has been received (This is the definition of an accepted tour).

Of course, this program can only estimate future cash receipts. To get a more accurate estimate of future cash receipts, this report can be run for periods past and then be compared against the previous estimates to get an idea of the relation between the estimates and the actual values.
