/PROG_HDR/

Program Description:

This program displays agency information.

/SCREEN1/

/SDESC1/

Program Operation:

Enter the agency arc number, or enter a starting agency name and zip code to search through the agencies.

All the agency information will display.

Note: This screen is more useful than booking agency entry when looking for agency information because, 1) It does not lock the agency record; 2) It does not start a booking.
