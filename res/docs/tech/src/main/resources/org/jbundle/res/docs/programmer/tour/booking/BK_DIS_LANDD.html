/PROG_HDR/

Program Description:

This program is used to display land by code.

/SCREEN1/

/SDESC1/

Program Operation:

Enter the starting land code to display, or press return to start at the beginning. If you want to see land pricing, enter a passenger count and a travel date for the price.

All the land will display in alphabetical order. Optionally, the Seat-in-car/coach and private car prices will also display.
