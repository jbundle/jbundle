Program Description:

This program maintains the transportation pricing information.

/SCREEN1/

/SDESC1/

Program Operation:

The transportation pricing file maintenance program is a simple file maintenance program.

First, type in the transportation code. The current costs will display.

For each line, enter the code for the transportation category (ie., 1=First Class, etc), the class description, the pax ranges and the cost.

Note: A maximum of four classes are allowed (numbered 1-4).
