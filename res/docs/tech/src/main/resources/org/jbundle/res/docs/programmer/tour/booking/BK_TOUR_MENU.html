/PROG_HDR/

Program Description:

This is the main screen for maintaining all header information for all the tours.

Program Operation:

/SCREEN6/

/SDESC6/

First, enter the tour code of the tour to add or change. All of the current headers for this code will display. Then, enter the starting date of the tour header to edit. If a header with this date doesn't exist, then you will be transfered to the Tour Header entry module automatically to enter the information for a new tour header.

/SCREEN8/

/SDESC8/

Next, all of the existing questions set up for this header will display. Select a question sequence number.

/SCREEN9/

/SDESC9/

Now the answers for the selected question will display. Select an answer sequence.

/SCREEN10/

/SDESC10/

The Tour Header Maintenance Menu displays automatically. Select any of these options to modify information.

/SCREEN11/

/SDESC11/

Note: Item number 3 (Tour Departure Control) works only with Group Series Tours. Items 6 thru 12 require valid Question/Answer sequence numbers to correctly use the information entered. FIT Component headers can have only ONE Question/Answer sequence--Question/Answer selection does not work for FIT Component headers. Pricing can not be entered for FIT component headers.
