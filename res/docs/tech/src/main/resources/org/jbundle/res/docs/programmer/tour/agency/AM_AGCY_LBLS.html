/PROG_HDR/

Program Description:

This program prints out the agency file in label format.

/SCREEN1/

/SCREEN2/

Program Operation:

First, enter a beginning and ending zip code. Then enter a "Y" or "N" under the "Exclude types?" question. If you decide to exclude types, then all the agencies will print EXCEPT the types entered, otherwise only the types entered will print. Next enter affiliation codes to print. Only agencies with matching affiliations will print. Last, enter the number of labels across on the sheet.

The labels will print automatically.
