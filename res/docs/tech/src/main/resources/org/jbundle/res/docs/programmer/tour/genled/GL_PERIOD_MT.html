/PROG_HDR/

Program Description:

This module allows you to enter or change the accounting periods.

/SCREEN1/

Program Operation:

First Time Entry:

The first time this module is entered, the computer will prompt the user for the starting date of the first accounting period, otherwise the current accounting periods will display and you will have a chance to change them.

Note: The accounting periods will automaticly change after the year-end program has been run; therefore the periods will be changed very infrequently.
