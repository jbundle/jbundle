/PROG_HDR/

Program Description:

This program is used to display a transportation product's prices and/or enter the transportation info needed for FIT building.

/SCREEN1/

Program Operation:

The program operation depends on what information is passed in:

If the transportation code is not passed in, then the transportation code is entered, the file is read, and the transportation information is displayed.

If the date has not been passed in, then the date is entered. The date is used to read the correct transportation rate.

Next, the rates are checked to see if they exist. Each transportation class is displayed and you can select one from the list. If the prices don't exist, you can enter the amount per-person, or type zero to skip this transportation.
