/PROG_HDR/

Program Description:

This program clears old bookings and booking history from the files.

/SCREEN1/

Program Operation:

First, enter the booking departure cut-off date. Bookings with departures before this date will be deleted from the system.

Then, enter the booking history cut-off date. Booking history for bookings departing before this date will be deleted from the system. (Of course, booking history will have been deleted from any booking purged under #1)

WARNING: This program deletes information from the booking files. ALWAYS back-up the entire system before running this program.
