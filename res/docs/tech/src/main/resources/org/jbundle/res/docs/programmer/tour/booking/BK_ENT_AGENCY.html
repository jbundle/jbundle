/PROG_HDR/

Program Description:

This program is used to enter the booking's travel agent information.

/SCREEN1/

Program Operation:

If the agency has been entered already, then it will display.

First the program asks for the Agency ARC number.

If 0 is typed, then this is a manual entry, a direct booking, or an agency without an ARC number.

The agency may also be searched by zip code and agency's name.

If the agency isn't found by their ARC number, you will be given the chance to enter it.

Once the agent is displayed, the information can be changed (Agency file will automatically be updated).

If you type DEL at the ANY CHANGE prompt, then all the agency information will be cleared from this booking and you will be prompted for the ARC number again. This does not affect the agency master file.
