/PROG_HDR/

Program Description:

Auto-distribution code are used to simplify the distribution of a amount to several accounts in a pre-described percentage. (For example, you may want to distribute a cost to three profit centers in set percentages)

/SCREEN1/

Program Operation:

First, enter a Auto-distribution code.

Next, enter each account and the percentage to distribute. The total distribution must equal 100%.

To use an Auto-distribution code, just enter the code in the Auto-distribution field in the Chart-of-Accounts Record. When an entry is made with this account, you have the option of posting to the Auto-distribution accounts.
