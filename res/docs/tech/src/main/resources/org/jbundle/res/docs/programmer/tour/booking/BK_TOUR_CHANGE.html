Program Description:

This program sets up or changes a tour block size for a range of dates.

Program Operation:

First, enter the tour code. The tour name will display.

Now, enter the starting date for the block, the ending date for the block, the number of passengers to change the block to, and the number to oversell if applicable.

Make any changes at the "Change at item number:" prompt, then hit return. Typing DEL at the prompt will delete the block for the range of dates entered.

First, the system will read through all of the inventory detail within the date ranges to check to see that there is enough space available to change the inventory. Then the range is read again to set up the block.

Note: To close a range, enter -1 for the blocked passengers. The block will be reduced to the currently used passengers.
