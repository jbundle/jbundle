/PROG_HDR/

Program Description:

This program is used to display ticket information.

/SCREEN1/

Program Operation:

Either enter a booking number to search for a ticket, or press return and enter the ticket number or ticket prefix to search for.

All of the ticket information will display. Hit return to exit this program.

Note: Only in certain instances can this program be used to change ticket information; If the booking was cancelled or the tour has departed, there is no way to modify ticketing information in booking entry, so you can change the information here. Otherwise, you must use the ticket maintencance system in booking entry.
