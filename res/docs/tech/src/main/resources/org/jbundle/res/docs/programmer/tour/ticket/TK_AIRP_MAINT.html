/PROG_HDR/

Program Description:

This Program maintains the Airport/City codes file.

/SCREEN1/

Program Operation:

Enter the three letter ARC airport code (or if the city doesn't have an airport, make up a unique city code).

The first three description fields are used for different purposes. The itinerary description is used for describing the airport in an itinerary [ie., Los Angeles (LAX)]; The second is the description for a ticket [ie., LOS ANGELES-LAX]; The third is for describing the city in an itinerary [ie., Los Angeles].
