/PROG_HDR/

Program Description:

This program allows you to display and change the tour hotel information.

/SCREEN1/

Program Operation:

If any hotel detail already exists, then the current hotels display.

The hotels are entered line-by-line. First the hotel code is entered, then the number of nights, rate and class, inventory flag, tour order flag, acknowledgement days, and hotel status.

If you answer "Y" to the INV question, the current hotel block will display.

To enter meals, just enter a meal code and the system will prompt for the meal quanity and meal days (ie., 234 = Days 2,3,4).

Typing DEL at the "Change" prompt clears all the current hotels and reverts back to the original hotels for this tour.
