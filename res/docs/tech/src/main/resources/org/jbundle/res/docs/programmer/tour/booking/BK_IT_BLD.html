/PROG_HDR/

Program Description:

This program is used to create and edit an itinerary for a booking.

/SCREEN1/

Program Operation:

First, enter the name of the word processing file to use as a template.

Next, enter the itinerary paragraph version code. Then select the day and date formats as well as the departure/arrival flight format from the first two lists as the bottom of the screen.

If you want to immediately edit the itinerary after it is built, answer yes to the last question.

The itinerary will automatically be built. If any itinerary paragraphs are not found, or a manual entry is read, the system will ask if you want to enter the text now. If not, the text for the excluded paragraph will be left blank.

/USE: BK_IT_BUILD/
