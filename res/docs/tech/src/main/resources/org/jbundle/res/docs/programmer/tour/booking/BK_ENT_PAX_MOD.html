/PROG_HDR/

Program Description:

This program allows you to make modifications in the hotel, land, air, remark, or comment detail for a specific passenger.

/SCREEN1/

Program Operation:

First, all of the passengers will display. Enter the sequence number displayed next to the passenger to modify.

The passenger modification menu will display.

/SCREEN2/

Select the menu item to modify the information this passenger. Remember, if you modify any information for a passenger, all the header information will be disregarded. For example, doing a hotel modification on the first passenger means that this passenger (and the sharing passenger) will use the hotels entered, not the hotels in the main screen.
