/PROG_HDR/

Program Description:

This program is used to display all reservations made by an agency.

/SCREEN1/

Program Operation:

This program is handy when checking to see if agencies have booked or made reservations in the past.

Enter the agency ARC number of reservations to display. If you have a booking active, the ARC number defaults to the current ARC number.

Next, enter the starting booking date to display. Reservations display in booking date order.

Remember: All reservations are displayed including cancelled or bookings on request.
