/PROG_HDR/

Program Description:

This program is used to display and change the current voucher status for a tour.

/SCREEN1/

Program Operation:

Enter the tour number of the tour to check.

All the vouchers for the tour will display along with the voucher status.

To set a voucher for re-printing, select the item number and answer Yes to the reprint question. The status will be changed to: reprint voucher.
