/PROG_HDR/

Program Description:

This program prints a cash receipt confirmation for all of the checks in the current batch.

/SCREEN1/

/SCREEN2/

Program Operation:

The current batch name will display on the screen. To print the payment confirmations, hit return at the CHANGE ITEM prompt.

A confirmation will be printed for each cash receipt in the current batch. The confirmations should then be stamped and mailed.
