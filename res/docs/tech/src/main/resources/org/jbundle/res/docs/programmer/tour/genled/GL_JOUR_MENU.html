/PROG_HDR/

Program Description:

This program is the menu for entering, editing, and posting General Journal Entries.

/SCREEN1/

Program Operation:

At the Batch name entry, hit return to start or edit your batch. You can enter another name to edit someone elses batch.

Next, enter the default date, comment, and source code for transaction entry. These will be the default values for the journal entry program.

Next, select the program to run or hit return to go back to the G/L menu.
