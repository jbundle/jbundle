PROGRAM DESCRIPTION:

This program is used to enter or change the daily availability for a tour. The inventory entered here can be used in the booking program.

PROGRAM OPERATION:

After entering the tour number, the description of the tour with a ending date after the system date will display.

Next, you enter a starting date. The screen will automatically display all of the inventory starting with this date.

If you want to modify the inventory or add to the inventory, you can enter an ending date and the maximum number of passengers booked on these dates.

If an inventory hasn't been added for a particular date, one will be written. If one already exists, the availability will be reduced to the inventory amount (unless the number used is greater than the new inventory amount in which case, the inventory is reduced to zero.)

NOTE:

The INV_DET file is set up as follows: ENTRY(1,4)=Tour code; TYPE = "T"our; ROOM_TYPE = " ";DATE = date
