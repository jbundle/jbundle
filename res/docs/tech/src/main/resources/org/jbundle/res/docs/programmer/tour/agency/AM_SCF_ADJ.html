/PROG_HDR/

Program Description:

The SCF Adjustment screen is used to make changes in an SCF field for a range of zip codes.

/SCREEN1/

Program Operation:

Enter the starting and ending zip code and enter a value for either the Sales Region, Territory, or UPS zone. The value entered will be entered in the SCF file for the entire zip range selected. Leave the field blank if you don't want the field to cahnge for the range selected.
