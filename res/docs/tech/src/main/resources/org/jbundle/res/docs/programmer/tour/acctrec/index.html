Program Description:

The A/R menu program is used to select an A/R application.

/SCREEN1/

Display Programs

Program Operation:

Select the appropriate program from the menu.

Accounts Receivable is the system which keeps track all all incoming receipts. The receipts may be for tours or for non-tour related items. The receipts may be by cash or MCO.

This system is also used for all transactions related to bookings. This includes refunds and A/R adjustments.
