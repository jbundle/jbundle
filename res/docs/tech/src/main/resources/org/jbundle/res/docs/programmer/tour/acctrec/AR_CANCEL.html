Program Description:

This program is used to cancel overdue bookings.

----------------------------------------------------------------------

Program Operation:

First enter the "Past due as of" date. If a payment is past due at this date, then it will be included on the report.

The next two questions want to know if you want to check the deposit and final payment dates. Both of the answers default to yes.

The last two questions want to know if you want to print the report and the notices. The report is used if you want to call the agents about the past due amounts. Answering yes to notice printing will print the past due notices to be mailed out.
