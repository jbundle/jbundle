Program Description:

This program organizes and summarizes G/L detail by account number, then date for spreadsheet analysis.

/SCREEN1/

/SCREEN2/

Program Operation:

This report will always print in spreadsheet format. Spreadsheet format will not print, but will output the file into the :PRINT directory under the name SSHEET_XXXX_USERNAME. This file may be retrieved in standard spreadsheet format.

Enter the rate type and period length. For example, W and 2 will summarize the data on two-week intervals. Q and 3 will summarize the data quarterly. Enter a date range for the date type selected.

The spreadsheet will be created automatically.
