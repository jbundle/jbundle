/PROG_HDR/

Program Description:

This program is used to maintain the hotel cost information.

/SCREENX/

/SDESCX/

Program Operation:

Enter a hotel code a the prompt, and the current hotel prices will display.

If a hotel rate/class cost is exactly the same as another cost, leave all the information blank, except the date range and the "Use R/C" column. Enter the rate/class of the rate which is exactly the same.
