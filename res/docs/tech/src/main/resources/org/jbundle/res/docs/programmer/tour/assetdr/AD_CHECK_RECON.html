/PROG_HDR/

Program Description:

This program is used to mark transactions as reconciled per a bank statement.

/SCREEN1/

Program Operation:

Enter the account to reconcile. Then enter the Type of transaction (for example, "C" to reconcile all checks), press return for all types of transactions. Then enter the starting transaction number and the date the items should be marked as reconciled. If you want items that have already been reconciled to display, then answer Yes to the last question.

Now, each transaction will automatically display, one by one, and you will have a chance to enter "Y" to mark them as reconciled.
