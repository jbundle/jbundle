/PROG_HDR/

Program Description:

This program allows you to set up several different versions of your Financial Statements.

This is the menu program for Financial Statement setup.

/SCREEN1/

Program Operation:

Enter a financial Statement code. This is the code you will use to set up and print this set of financial statements (For example you could have a code: MGMT for the Managerial Financial Statements, a code FINL for the statements available to outsiders, and a code STMTC for Statement of Changes).

Then enter a short description of this version of the statements. Then enter the number to display. The valid numbers are: (T)o-date balance or (N)et change in period (ie., Statement of Changes).

After entering the code, you can select a program to set up and process the Financial Statements.

Note: The Financial Statements refer to a COMPLETE set of statements utilizing ALL of the accounts in the chart of accounts. The statements might contain an Income Statement, Statement of Retained Earnings, and a Balance Sheet. Or it might contain a Statement of Changes in Financial Position.
