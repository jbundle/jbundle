/PROG_HDR/

Program Description:

This program allows you to display and change the tour pricing information.

/SCREEN1/

Program Operation:

If the price already exists, then the current prices display.

The prices are entered line-by-line. First a Description is entered, then the unit price, quanity, and commission % (which defaults to the percentage entered in the agency screen) are entered.

Typing DEL at the "Change" prompt clears all the current pricing and reverts back to the original tour pricing.
