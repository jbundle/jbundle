/PROG_HDR/

Pprogram Description:

This program is used to enter the pricing for any kind of tour.

/SCREEN1/

/SDESC1/

Program Operation:

The tour header description and the current question/answer sequence will display.

Now, the current pricing will display.

The passenger category tells the booking system what to multiply the price by in a booking (ie., 0 - for each passenger, 1 for each single room, etc).
