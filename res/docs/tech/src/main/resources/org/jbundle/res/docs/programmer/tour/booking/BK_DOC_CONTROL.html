/PROG_HDR/

Program Description:

This program is used to check and change the status of automatic updates.

/SCREEN1/

Program Operation:

Enter the tour number to check (defaults to the tour for the current booking). The regularly scheduled documents for this tour will display. Next to each number is a flag to indicate whether the update has run already. If the update has run already, then the flag will be set to Yes.

At the CHANGE ITEM prompt, you can reschedule any of the automatic prodedures to re-run by entering a Yes by the procedure to re-run. You can also enter the reference of the manual file for this tour.

Procedures may not be unscheduled to run.

The tour status may be changed except when tour is for a non group series tour and the current booking is not the booking for this tour.

Note: TOUR_STATUS - The valid tour status' are as follows:

OK - OK tour (Services can be arranged or confirmed)

GB - Group Blocked

CL - Tour closed to more bookings (type G only)

XL - Tour canceled

RQ - Non OK status

NS - No status (No operations will be done on this tour)

(ie., proposal status)
