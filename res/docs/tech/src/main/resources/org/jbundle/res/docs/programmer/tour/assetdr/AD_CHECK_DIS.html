/PROG_HDR/

Program Description:

This program allows the review of a range of checks or to inquire as to an account balance at a date.

/SCREEN1/

Program Operation:

Type in the reference number of the account to display. Then type in the starting date to display transactions.

Next, if you would like to see the starting and ending balance for your range, enter "Y". All checks, deposits, etc will display automatically. If you selected account balance displays, the beginning balance will display, then after displaying the detail, the ending balance will display. The account balance display will take more time as the beginning balance at a date must be calculated.

Note: Because of the sensitivity of payroll data, payroll checks will display, but the amount will be blank. To get the amount, use the payroll system.
