Program Description:

This program is used to maintain the passenger file.

/SCREEN1/

Program Operation:

The passenger file is mainly used for follow-up mailing to clients who have booked tours in the past. This file can also be used to maintain any list of names and addresses.

First, enter the passenger number to retrieve. To enter a new passenger, type a zero or a non-existent member number. To search by last name, press return at the member number field and enter the first few characters of the last name to match.

If you are entering a new passenger, and there is an active booking on the screen, the first four passengers are displayed and the system sets up a default for the name, saluatation, and booking number fields.

Later, the passengers can be retrieved by travel interest code, passenger type, or any information retrieved from the associated booking.
