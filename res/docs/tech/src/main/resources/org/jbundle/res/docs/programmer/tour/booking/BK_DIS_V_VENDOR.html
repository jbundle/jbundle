/PROG_HDR/

Program Description:

This program is used to display the current voucher status for a vendor.

/SCREEN1/

/SDESC1/

Program Operation:

Enter the vendor code of the vendor to display. Next, enter the starting service date to display. Then answer the question, "Do you want to display acknowledged vouchers also?"

All the vouchers for this vendor will display in order of the date the first service is to be provided.

Note: If you display acknowledged vouchers only, there may be a delay in the display. The system still has to read thru all the vouchers for this vendor.
