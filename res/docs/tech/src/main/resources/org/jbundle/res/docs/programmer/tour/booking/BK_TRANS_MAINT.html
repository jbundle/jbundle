Program Description:

This program maintains the transportation file.

/SCREEN1/

/SDESC1/

Program Operation:

The transportation file maintenance program is a simple file maintenance program.

First, type in the transportation code. It is usually coded like this:

First 3 letters: City code of Departure City

Next 3 letters : City code of Arrival City

Next letter : Type of transportation (ie., R-Rail)

Next letter : Vendor initial if applicable

For example, the code: TYOOSAR would be rail from Tyoko to Osaka.

The country and city codes should be the ARC airport codes, or three letter abbreviation of the city if there is no airport.
